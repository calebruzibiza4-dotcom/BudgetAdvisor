# Budget Advisor

A premium budget advisor app with a React + Vite client and an Express server.

## Structure

- client/: React + Vite frontend
- server/: Express API server

## Run locally

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run start
```

The Vite client proxies API requests to the server at http://localhost:3000.
